Juice-Shop Windows平台搭建
网络安全 windows
windows 2018-02-26

Juice-shop是OWASP的一款漏洞环境，里面包括了目前的TOP10漏洞，还有一些有深度的漏洞，难度各有不同，juice-shop可以用docker来部署也可以在windows或linux的环境下用node.js发布出来，下面就开始搭建

1.首先在github上下载源码
https://github.com/bkimminich/juice-shop/releases/tag/v6.4.2

下载自己对应的系统以及node版本的文件，要注意juice-shop-6.4.2后面跟着的这个node，如果是node6那么你就需要去按照对应的node.js为6.x版本的才能成功发布出来（node.js下载可到https://github.com/nodejs/Release）

2.安装node，打开你下载的node安装包，安装时全部默认即可，然后先检验node.js是否安装成功
Win+r打开运行输入cmd开启命令行，再打node –v查看node版本如果可以看到版本且是自己需要的版本说明安装无误

3.发布Juice-Shop
打开CMD，使用cd命令来进入Juice-shop源码文件夹（也可在文件夹中Shift+鼠标右键，点击“在此处打开命令窗口”)
然后输入nmp start

完成
用浏览器访问本地3000端口即可看到页面
