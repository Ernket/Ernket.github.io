Juice-Shop Windowsе№іеҸ°жҗӯе»ә

Juice-shopжҳҜOWASPзҡ„дёҖж¬ҫжјҸжҙһзҺҜеўғпјҢйҮҢйқўеҢ…жӢ¬дәҶзӣ®еүҚзҡ„TOP10жјҸжҙһпјҢиҝҳжңүдёҖдәӣжңүж·ұеәҰзҡ„жјҸжҙһпјҢйҡҫеәҰеҗ„жңүдёҚеҗҢпјҢjuice-shopеҸҜд»Ҙз”ЁdockerжқҘйғЁзҪІд№ҹеҸҜд»ҘеңЁwindowsжҲ–linuxзҡ„зҺҜеўғдёӢз”Ёnode.jsеҸ‘еёғеҮәжқҘпјҢдёӢйқўе°ұејҖе§Ӣжҗӯе»ә

1.йҰ–е…ҲеңЁgithubдёҠдёӢиҪҪжәҗз Ғ
https://github.com/bkimminich/juice-shop/releases/tag/v6.4.2

дёӢиҪҪиҮӘе·ұеҜ№еә”зҡ„зі»з»ҹд»ҘеҸҠnodeзүҲжң¬зҡ„ж–Үд»¶пјҢиҰҒжіЁж„Ҹjuice-shop-6.4.2еҗҺйқўи·ҹзқҖзҡ„иҝҷдёӘnodeпјҢеҰӮжһңжҳҜnode6йӮЈд№ҲдҪ е°ұйңҖиҰҒеҺ»жҢүз…§еҜ№еә”зҡ„node.jsдёә6.xзүҲжң¬зҡ„жүҚиғҪжҲҗеҠҹеҸ‘еёғеҮәжқҘпјҲnode.jsдёӢиҪҪеҸҜеҲ°https://github.com/nodejs/Releaseпјү

2.е®үиЈ…nodeпјҢжү“ејҖдҪ дёӢиҪҪзҡ„nodeе®үиЈ…еҢ…пјҢе®үиЈ…ж—¶е…ЁйғЁй»ҳи®ӨеҚіеҸҜпјҢз„¶еҗҺе…ҲжЈҖйӘҢnode.jsжҳҜеҗҰе®үиЈ…жҲҗеҠҹ
Win+rжү“ејҖиҝҗиЎҢиҫ“е…ҘcmdејҖеҗҜе‘Ҫд»ӨиЎҢпјҢеҶҚжү“node вҖ“vжҹҘзңӢnodeзүҲжң¬еҰӮжһңеҸҜд»ҘзңӢеҲ°зүҲжң¬дё”жҳҜиҮӘе·ұйңҖиҰҒзҡ„зүҲжң¬иҜҙжҳҺе®үиЈ…ж— иҜҜ

3.еҸ‘еёғJuice-Shop
жү“ејҖCMDпјҢдҪҝз”Ёcdе‘Ҫд»ӨжқҘиҝӣе…ҘJuice-shopжәҗз Ғж–Үд»¶еӨ№пјҲд№ҹеҸҜеңЁж–Үд»¶еӨ№дёӯShift+йј ж ҮеҸій”®пјҢзӮ№еҮ»вҖңеңЁжӯӨеӨ„жү“ејҖе‘Ҫд»ӨзӘ—еҸЈвҖқ)
з„¶еҗҺиҫ“е…Ҙnmp start

е®ҢжҲҗ
з”ЁжөҸи§ҲеҷЁи®ҝй—®жң¬ең°3000з«ҜеҸЈеҚіеҸҜзңӢеҲ°йЎөйқў

ж„ҹи°ў@buttersеҜ№иҜҘж–Үз« зҡ„жҠҖжңҜж”ҜжҢҒгҖӮиҪҜд»¶й“ҫжҺҘең°еқҖеқҮдёә@buttersжҸҗдҫӣгҖӮеҰӮйңҖиҪ¬иҪҪпјҢиҜ·жіЁжҳҺеҮәеӨ„гҖӮеҗҰеҲҷе°ҶиҝҪз©¶жі•еҫӢиҙЈд»»
