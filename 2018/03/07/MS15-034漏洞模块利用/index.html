MS15-034漏洞模块利用

更新源

在kali中自带的msf由于版本太低，有的模块是没有的，所以我们先更新源

源的文件位置:/etc/apt/sources.list
下面是更新源:
官方的：deb http://http.kali.org/kali kali-rolling main non-free contrib
中科大：deb http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib
deb-src http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib
将上面的源添加进/etc/apt/sources.list后，执行apt-get update即可
更新完后，执行apt-get install metasploit-framework更新msf

更新完后，输入msfconsole打开软件

可以看到已经更新为v4.16.32了

查找/利用模块

进来后，不清楚ms15-034的利用模块在哪，我们可以执行search ms15-034来查找

然后use auxiliary/scanner/http/ms15_034_http_sys_memory_dump后，show options来查看需要配置的

设置好对方ip及目标url后，exploit执行
