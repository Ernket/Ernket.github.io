#!/usr/bin/python
# -*- coding: UTF-8 -*-
a = 9
b = 2

c等于a除b的余数，就相当于9/2等于8余1，那么c就等于余数1

#!/usr/bin/python
# -*- coding: UTF-8 -*-
a = 2
b = 3
c = a**b 
print "a的b次方的值为：", c

这个就很好理解啦,a**b，就相当于a^b，也就是a的b次方

#!/usr/bin/python
# -*- coding: UTF-8 -*-
a = 9
b = 2
c = a//b 
print "a除b的商取整数的值为：", c

//和%是反着来的，%取的是余数，而//取的是整数，a=9,b=2,a//b就等于4.5，取整数，就变成了4，于是a//b的输出结果就是4

3月20号:Python字符串(版本3.x)

字符串基本操作

字符串的基本操作包括格式化，转义字符，格式化符号，字符串方法。
不多说废话，现在开始
字符串格式化

>>>print("Elapse and $s"%'Asolmn')
Elapse and Asolmn

>>>print("我已经%s岁了"%10)
我已经10岁了

以上就是字符串的格式化，%这个符号代表的是占位符，%后面跟的是一个希望格式化的值，然后右边是格式化的值，这个值可以是数字也可以是字符串。
除了这些格式化之外还有元组的格式化

>>>print("我是%s,已经%d岁了"%('Asolmn',17))
我是Asolmn,已经17岁了

这是元组的格式化，右边的%跟的这个东西就是元组。

3月20日:Python条件语句

Python中的条件语句，是通过一条或者多条的语句的执行结果（true或false）来决定执行的代码块，Python中if语句就是用来控制程序的执行，作一个判断的效果:

if (判断条件):
    如果是true，执行这条语句
else:
    如果返回false，执行这条语句

那么下面就来举个例子

a = 1
b = 2
if a != b:
    print "a不等于b"
else:
    print "a等于b"

输出结果为a不等于b，它会根据你的判断条件，也就是if后面的语句，来判断返回的结果为true还是false，如果是true，就接着下面的一条执行，如果返回的是false，则执行else后面的语句。
扩展：多个判断条件
当判断条件为多个时，可以用这种形式：

a = 2
b = 2
if a != b:
    print "1"
elif a == b:
    print "2"
else:
    print "3"

While循环语句

While语句用于循环执行程序，形式为:

while 判断条件:
    执行语句

当返回的值为true时，会一直循环执行，当判断条件为false时，就循环结束，举个例子

asolmn = 0
while (asolmn < 9):
    print "The num is:", asolmn
    asolmn += 1
print "Bye!"

输出结果为

The num is: 0
The num is: 1
The num is: 2
The num is: 3
The num is: 4
The num is: 5
The num is: 6
The num is: 7
The num is: 8
Bye!

当”asolmn”小于9时，返回就一直是true，所以便会一直执行下面那一条输出语句，接着”asolmn”的值+1，返回到第一条，继续执行，直到返回false

3月23日:Python Break语句

Break语句，事用来终止循环语句的，即没有返回false或者递归完也会停止循环的语句。
例子：

for elapse in 'asolmn':
   if elapse == 'o':
      break
   print 'test:', elapse

一个for是循环了一个字符串，当他要循环时，他会经过一个if判断，判断elapse等不等于’o’，若等于，就打破循环（停止循环），并执行下一条

4月6日:Python日期和时间

在Python中的time模块下有很多转换常见日期格式，time.time()能获取当前时间戳

import time
eee = time.time()
print "timestamp:",eee

输出结果为

timestamp: 1523002846.01

时间戳适合做日期运算，但是1970年之前的就无法表示了，太遥远的也不行，Unix、Windows只支持2038年

可以使用函数，只要将浮点数进行传递，例如localtime等函数

import time
eee = time.localtime(time.time())
print "the time:",eee

输出结果为

the time: time.struct_time(tm_year=2018, tm_mon=4, tm_mday=6, tm_hour=16, tm_min=24, tm_sec=9, tm_wday=4, tm_yday=96, tm_isdst=0)

5月17日:continue语句

continue和break都是跳出循环的语句，但是两者的不同点在于，continue是跳出本次循环，而break是跳出整个循环，执行其他的

例子:

for elapse in 'Elapse':
    if elapse == 'a':
        continue
    print "The current is:", elapse

print "have fun!"

输出结果:

The current is: E
The current is: l
The current is: p
The current is: s
The current is: e
have fun!

6月14日 exec执行函数

注意： 在 Python2 中exec不是函数，而是一个内置语句(statement)，但是Python 2中有一个 execfile() 函数。可以理解为 Python 3 把 exec 这个 statement 和 execfile() 函数的功能够整合到一个新的 exec() 函数中去了。
所以类似功能的函数在python2中是execfile。

适用版本: 3.x

语法

exec(object[, globals[, locals]])

参数解释

object：必选参数，表示需要被指定的Python代码。它必须是字符串或code对象。如果object是一个字符串，该字符串会先被解析为一组Python语句，然后在执行（除非发生语法错误）。如果object是一个code对象，那么它只是被简单的执行。

globals：可选参数，表示全局命名空间（存放全局变量），如果被提供，则必须是一个字典对象。

locals：可选参数，表示当前局部命名空间（存放局部变量），如果被提供，可以是任何映射对象。如果该参数被忽略，那么它将会取与globals相同的值。

返回值

exec 返回值永远为 None。

实例

>>> exec 'print"hello world!"'
hello world!
#执行单行语句
>>> exec """for i range (5):
    print 'the number is:',i
    """
the number is: 0
the number is: 1
the number is: 2
the number is: 3
the number is: 4
#执行多行语句
