前言

这个靶机难度中等，主要是后期进入靶机后提权的操作，前期只要留心，基本上都能操作出来

下载地址

渗透

网站内容没有什么提示，网站主题我挺喜欢的，开启抓包来枚举一下子目录

发现了一个网站备份.bak,查看源代码，找到用户名和密码哈希，丢进工具里破解

root@Elapse:~# john pass.txt

密码出来了，frank!!!，但是找不到地方登录，想起还有一个8011端口，打开发现只有一串大字

Development Server !

用dirb扫到了有/api/这个路径，访问看看

意思就是说，这个网站还有以下功能，但是大部分都还没有完成，一个一个访问后发现，只有files_api.php是可以连接的

这里的意思是说，要给它一个参数，经过几次尝试后发现参数为file，用curl来访问一下

存在任意文件读取的漏洞，先把目光转移到主页，同样用dirb来扫一下别的目录

root@Elapse:~# dirb http://192.168.1.108

找到了一个burp suite没有显示出来的路径，/development，打开来看看，发现要登录，把先前找到的用户密码输进去，成功，以下是网站内容

* Here is my unfinished tools list
- the uploader tool (finished but need security review)

意思是，这网站有一些未完成的工具，前面可以猜出，这里应该有个上传点
那我们就来猜一下路径
试了很多路径，最后发现上传路径是/development/uploader/上传点get，接着生成一个木马触发一下就好了

这里要注意的是，直接上传php木马的话，会给拦截，所以要绕过，在PHP的前面加一个GIF98，然后后缀改为gif就可以上传了，然后路径还需要自己爆破

用wget的方式下载到靶机里去，gcc编译，./运行，提权成功
