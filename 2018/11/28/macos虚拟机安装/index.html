虚拟机安装MacOS
linux
linux 2018-11-28

环境

  • Vmware Workstation任意版本
  • MacOS 10.11(下载地址:链接：https://pan.baidu.com/s/1iYLYVelaBaJuCpwCqkUhBg 提取码：c8y5 )
  • unlocker(下载地址：链接：https://pan.baidu.com/s/1Rr_4K1Z4UZ2xU44kGRCNzg 提取码：rsgk ）

操作

Vmware安装懒得说，主要讲系统安装
Vmware是没有安装mac系统这个选项的
所以需要unlocker
解压了之后，管理员方式运行win-install.cmd
安装完后打开Vmware后就可以看到这个选项了

在下面选择系统版本为10.11，然后下一步（后面的默认即可）

这里要注意一下，直接启动虚拟机会报错
所以你需要通过编辑vmx文件，添加一条配置来解决
打开虚拟机的vmx文件

找到smc.present = "TRUE"
在下面添加一行smc.version = 0

重新打开macOS虚拟机，这个时候就不会报错了
接着你会来到这个界面

选择什么语言看你喜欢，反正我用中文
继续下一步，他会让你选择安装的位置，但是它告诉你没有足够的空间来安装

当时我还以为真的是我设置的40G少了，后来看了看觉得不对劲，这个不是安装的系统吗，怎么可能有内存安装
正确的操作是
点击实用工具-磁盘工具

点击你的虚拟磁盘，然后格式化


单机抹掉，完成格式化

然后我们回到之前的页面，可以看到多出来了一个Elapse这个盘

点继续，等待安装

安装完毕后，就是简单的系统设置了

根据自己需要设置

设置完后，就完事了
